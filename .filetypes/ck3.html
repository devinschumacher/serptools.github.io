What is a CK3 file?

A CK3 file is a saved game created by Paradox Interactive Crusader Kings III. It contains a record of a player's progress during a Crusader Kings III game. Some CK3 files are plain text files, while others are .ZIP files that contain a plain text record of a player's saved game.

More Information

Crusader Kings III is a strategy game in which players control a dynasty during the Middle Ages. As the game progresses toward the year 1453, players must earn prestige and continue their dynasty for as long as possible.

When Crusader Kings III saves a player's current game, it stores the saved game data in a CK3 file. If a player manually saves their progress, Crusader Kings III creates a Zip-compressed CK3 file. This archive contains a compressed plain text record of the player's game state. When Crusader Kings III autosaves a player's progress, the resulting CK3 file is saved as an uncompressed plain text file.

Plain text CK3 files, and the plain text gamestate files that compressed CK3 files contain, are comprised of several sections of editable saved game data and metadata. To learn more about these sections and how to edit them, click here.

How to open a CK3 file

You can open and edit plain text CK3 files with Microsoft Notepad, Apple TextEdit, or any other text editor. If your CK3 file is Zip-compressed, you must decompress it (with Apple Archive Utility, 7-Zip, or another Zip-decompression tool) to access the gamestate file it contains. Then, you can open and edit the gamestate file with any text editor.

(To use a gamestate file with Crusader Kings III, you must either repackage it within a compressed CK3 file or rename the gamestate file to use the .ck3 extension.)

How to use CK3 files with Crusader Kings II

To use a CK3 file with Crusader Kings III, place it in the game's save games directory. This directory is usually located at:

~/​Paradox Interactive/​Crusader Kings III/​save games

Then, the next time you start Crusader Kings III, you'll be able to load the saved game your CK3 file contains.
