WN Web File

Developer WN
Popularity
2.2  |  5 Votes
 

What is a WN file?

Web file used by WN, a Web server designed to protect visibility and access to server files; stores information about files served on the server; may describe .HTML files, images, and other binary files; used for specifying the content of Web files in a directory.

More Information

A WN file is compiled into the file index.cache, which is placed on the server and provides access to server files via HTTP requests.

WriteNow Document

Developer NeXT
Popularity
1.7  |  3 Votes
 

A WN file may also be a text document created by NeXT WriteNow, a word processor available for the original Apple Macintosh PC and NeXT workstations. WN files often contain instructional manuals, worksheets, resumes, memos, or letters. These documents typically consist of several sections of formatted text, and they may include images.

More Information

WriteNow was a word processor developed by John Anderson and Bill Tschumy and released by NeXT in 1985. Initially, WriteNow was available for the Apple Macintosh, as a competitor to Apple's MacWrite word processor. When NeXT began selling their own workstations, they bundled WriteNow with those workstations.

In WriteNow, users could create formatted text documents, complete with headers and footers, bolded, italicized, and underlined text, and images. On Macintosh computers, WriteNow saved documents without a file extension. However, on NeXT workstations, WriteNow saved documents with the .wn file extension.

How to open a WN file

WriteNow is discontinued, but you can open WN files with LibreOffice Writer, a free word processor included with the LibreOffice suite. To do so, select File → Open... from Writer's menu bar. Then, navigate to and open your WN file.

How to convert a WN file

You can use LibreOffice Writer to convert a WN file to the following formats (and several others):

  • .ODT - OpenDocument Text Document
  • .DOCX - Microsoft Word Document
  • .DOC - Microsoft Word Document (Legacy)
  • .HTML - Hypertext Markup Language
  • .TXT - Plain Text

To do so:

  1. Open your WN file with LibreOffice Writer.
  2. Select File → Save As...
  3. Choose the format to which you want to convert your WN file.
  4. Click Save.
