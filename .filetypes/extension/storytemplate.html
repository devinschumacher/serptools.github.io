What is an STORYTEMPLATE file?

A STORYTEMPLATE file is a project template created by Articulate Storyline 360, a program for developing e-learning content, such as interactive training courses and presentations. It is saved in the same format as a .STORY file, storing all the elements of a course, such as slides, animations, quizzes, text, video, audio, user interactions, and metadata. While STORYTEMPLATES are saved in the same format as STORY files, they are used to store learning courses that Instructional designers, corporate trainers, educators, etc., want to replicate with the same content and appearance.

More Information

Users typically work with STORY files during the course development phase since that is the default format for storing projects. However, if they want to replicate a course structure, they can save it as a STORYTEMPLATE file by selecting File → Save As, and then choosing "Storyline Template" from the "Save as type" dropdown menu.

For example, a university professor creating an online course might use STORYTEMPLATE files to develop multiple self-paced learning modules with a similar appearance. Once finalized, the professor can publish the STORYTEMPLATE file in formats like Sharable Content Object Reference Model (SCORM) or Experience API (xAPI) for use in a Learning Management System (LMS) such as Blackboard or Moodle.

How to open an STORYTEMPLATE file

You can open STORYTEMPLATE files with Articulate Storyline 360 by selecting File → Open. While Articulate 360 is available as an online tool, only the Windows version of the software (available within the Articulate 360 suite) supports STORYTEMPLATE files.

Like STORY files, STORYTEMPLATE files are Zip-compressed, which means you can open them with a compression utility, such as Microsoft File Explorer (bundled with Windows), Apple Archive Utility (bundled with macOS), and Corel WinZip (Windows). However, you may need to change the .storytemplate file extension to .zip for the utility to recognize it. Also, opening the file as a .ZIP archive will not appear the same as opening it with Articulate Storyline 360.
