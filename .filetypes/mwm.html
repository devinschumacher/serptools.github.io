Space Engineers 3D Model File

Developer Keen Software House
Popularity
3.9  |  9 Votes
 

What is an MWM file?

An MWM file contains a 3D model used by Space Engineers, a sandbox game used to engineer and construct spaceships to explore and survive space and planets. It stores 3D object appearance information, such as the vertices, length, and shininess. MWM files are used for various types of objects in the game, which include characters, tools, weapons, and spaceships.

More Information

You will most likely only encounter an MWM file if you play Space Engineers and are looking to modify objects in the game. Models can be found in the following directory:

\​Steam\​SteamApps\​common\​SpaceEngineers\​Content\​Models

MWM files store appearance information for various types of 3D objects, such as machine guns, angle grinders, automatic rifles, and hand drills.

Maps.me Map

Developer Maps.me
Popularity
3.0  |  1 Vote
 

An MWM file may also be a map used by Maps.me (formerly MapsWithMe), a navigation app for Android and iOS, and its variants. It contains a record of a region's streets, buildings, and/or natural features. Users download MWM files so they can use MapsWithMe and its variants while offline.

More Information

Maps.me is a mobile app that allows users to map walking, biking, or driving routes while offline. To create these routes, Maps.me uses map data saved in local MWM files (and route data saved in MWM.ROUTING files.)

The map data that MWM files contain is sourced from OpenStreetMap, a collaborative database that contains a detailed planetary map. However, MWM files are saved in a proprietary format, and are not the same as OpenStreetMap .OSM files. Thus, you can open and use MWM files only with Maps.me and forked versions of the app, such as Organic Maps.

Each MWM file is named for the regional map it contains. For example, an MWM file that contains data used to load a map of northern Italy would likely be named Italy_North.mwm.

How to open an MWM file

You can open an MWM file with Maps.me (Android, iOS) or Organic Maps (Android, iOS), provided you can access the directories from which the apps load map data. However, in most versions of Android and iOS, users can no longer access the directories from which Maps.me and Organic Maps load map data. Thus, you cannot manually open MWM files.

If you can access the directories from which Maps.me and Organic Maps load map data, you can open an MWM file by replacing the current version of that file with your MWM file. For example, if your MWM file contains a map of Denmark, and you want to use it with the Android version of Maps.me, place the file in your device's ~/​Maps.me/​Denmark directory (and ensure the file is named Denmark.mwm.) Then, Maps.me will use the new MWM file when loading map data.
