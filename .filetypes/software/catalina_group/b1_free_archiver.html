Software Overview

B1 Free Archiver is a simple file manager and archiver that allows users to create and expand archive files (such as .ZIP files). It is free, reliable, and available for Windows, Linux, Mac, and Android.

B1 allows users to decompress archives saved in most popular archive formats, including ZIP, .GZIP, .RAR, .7Z, .BZIP2, .TGZ, .TAR, .LZMA, .JAR, .CAB, and many more. It also allows users to create archives saved in B1's unique archive format, .B1, as well as ZIP, 7Z, TAR, GZIP, and BZIP2.

While B1's simplicity is its claim to fame, it does include a few advanced features. The program allows users to create password-protected archives, which are encrypted using a 256-bit AES algorithm, and split archives. It also allows users to view and edit archives' contents without decompressing them.

Summary

If you are looking for a free and easy-to-use archiver, B1 Free Archiver is worth a try. It will let you decompress almost any archive you come across and quickly create your own compressed archives.
