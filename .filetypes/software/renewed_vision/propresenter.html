Software Overview

Renewed Vision ProPresenter is an application used to create and show media presentations. It is available for Windows and OS X.

ProPresenter is built to display high quality productions of events such as sporting events, worship gatherings, and conferences. The application allows you to customize the appearance of your presentation, whether it be adding transitions, text fly-ins, or video effects. You can rip DVD video into the presentation or load video files, such as .MOV and .WMV.

The application uses proprietary file formats for playlists, templates, and DVD clips, that allow you to conveniently save and share presentation elements. ProPresenter also provides dynamic features such as an iPad remote, stage displays, cloud space for file synchronization, and a module for master control of multiple outputs.

ProPresenter is a premiere presentation application, as it’s used in many different professional venues to create dynamic experiences. Whether you need software for conference presentations, concerts, sporting events, or worship services, go with Renewed Vision ProPresenter.
