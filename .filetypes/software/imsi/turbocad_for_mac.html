Software Overview

IMSI TurboCAD Mac Deluxe is a CAD application used for drafting and modeling 2D and 3D designs. The tool is used in educational, home, and professional environments. The software is also available for Windows users as TurboCAD.

TurboCAD Mac Deluxe provides an array of tools for creating 2D and 3D illustrations, floor plan layouts, and diagrams. It features layers and layer sets for easier design editing and organizing, a multi-step drawing wizard for quick set up of your drawings, custom toolbar creation for instant access of your most often used tools, and markup and redlining tools for objects. The application also offers a set of surface analysis tools that help maintain design integrity for identifying surface irregularities.

The CAD tool supports many popular design file formats, including .PSD, .IGS, .JPG, .PNG, .3DS, .DWG, and .DXF. When you finish a design, there are several useful tools that help you publish your work. The Print by Layer Tool allows you to print specific layers of your multiple sets of data. You can also save designs as single or multipage PDFs to send through email.

TurboCAD Mac Deluxe is a quality design application that can be used for personal or professional purposes, whether it be a building layout or mechanical part. If you are looking for a reasonably priced alternative to Autodesk AutoCAD, IMSI TurboCAD Mac Deluxe is a competent application.
