Main Features

  • View, edit, and convert images saved in over 100 formats
  • Perform complex image processing tasks, including creating animations, from the command line

Software Overview

ImageMagick is a free, multiplatform program used to view, edit, and convert images. It supports over 100 image formats, including most common formats and many obscure ones.

While ImageMagick can be somewhat useful for basic users, especially those who want to open and convert images saved in obscure formats, it's most useful for advanced users, who can take advantage of the program's collection of command line tools. From the command line, advanced users can quickly perform both simple and complex image processing tasks, by specifying one or more commands and accompanying options. (For a complete list of ImageMagick's command line tools, click here.)

Summary

If you're looking to view or convert an image saved in an obscure format, or you're looking to perform advanced image processing tasks from the command line, ImageMagick might be right for you. If neither of those features appeals to you, you might be better off with a simpler image viewer or editor, such as XnViewMP.
