Software Overview

GRAPHISOFT Archicad is a building information modeling (BIM) application for Windows and macOS. Architectural professionals commonly use the software to design, visualize, collaborate, and document when engineering the interiors and exteriors of buildings, urban areas, and other structural environments.

Archicad can import and export various CAD formats, such as .DWG, .IFC, .DXF, and its proprietary file types, including the .PLN and .PLP projects. The application provides many tools for designing and visualizing structures, including 2D drawing tools, a 3D modeling interface, architectural rendering capabilities, and a document management tool with a data storage server. Archicad also allows you to add functionality to the application through free and commercial extensions.

The application provides helpful collaboration features for team members to work together. GRAPHISOFT's BIM Server is a real-time collaboration environment and the BIMcloud, which allows BIM workflows to be accessible on any software, hardware, or network configuration. You can also document your buildings with tools that comply with local BIM requirements.

Summary

Archicad is a leading BIM application in various architectural and engineering fields. It gives you many features for BIM, 3D visualizations, project management, working with teams, and engineering integration. These capabilities and the software's extensibility make GRAPHISOFT Archicad one of the leading BIM applications for professional architects.
