Software Overview

Cube 2: Sauerbraten is a first-person shooter (FPS) video game sequel to Cube, which was released in 2001. Much of the game was developed by Wouter "Aardappel" van Oortmerssen but other persons have contributed, as It is an open source project.

The game features quick and intense gameplay similar to Doom 2 and Quake 1. You can choose from several single player and multiplayer game modes and edit maps as you play. The game also features an online community of users that create and share maps, mods, and scripts.

Cube 2: Sauerbraten provides fun gameplay that Doom 2 and Quake 1 players will enjoy. The game is friendly to beginners as it provides helpful documentation and tutorials on how to play. Cube 2: Sauerbraten is a fun and intense FPS game.
