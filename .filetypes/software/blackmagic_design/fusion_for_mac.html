Software Overview

Blackmagic Design Fusion, previously Eyeon Fusion, is an advanced compositing application used to create visual effects and motion graphics. The software provides a node-based interface and a wide range of features and effects. It is available for free for Windows and OS X and for purchase as Fusion Studio with more features for Windows, OS X, and Linux.

Fusion features integrated 2D and 3D motion graphics and visual effects compositing capabilities. The application supports a wide variety of 3D model, audio, and video formats such as QuickTime Movies and Audio, JPEG, Photoshop, Alias Pix, 3D Studio 3DS, AutoCAD DXF, SVG, Cineon, Autodesk FBX, and Collada DAE. The software provides a large range of tools such as animation, advanced key framing, titling, GPU acceleration, and paint.

The software is used by visual effects artists, 3D animators, and broadcast and motion graphic designers to create TV commercials, title sequences, and broadcast graphics. It has also been used to create visual effects in many Hollywood film productions including Gravity, Edge of Tomorrow, Avatar, and Skyfall. The application is available for free which is great for trying it out to see if you'd like to purchase (at a reasonable price) the more powerful studio version. Blackmagic Design Fusion is an excellent choice for handling visual effects and motion graphics in your next blockbuster production.
