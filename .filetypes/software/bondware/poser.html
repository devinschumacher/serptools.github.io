Software Overview

Bondware Poser is a three-dimensional character design program that allows designers to create realistic 3D characters and animations. The software includes a large content library of predesigned human and animal figures, as well as accessories, such as props, hair, and clothing. The program also supports custom lighting and can import motion capture and sound files that can be applied to animations. Poser was originally developed by Smith Micro, but was sold to Bondware in June 2019.

Poser can be used for rapid 3D character design. For example, users can drag and drop pre-rigged figures from the content library to add them to a scene. Users can automatically add movements with the WalkDesigner tool, which animates a character's walking cycles. The TalkDesigner tool syncs mouth, eye, and head motions with sound files. Character accessories, such as clothing, may be automatically scaled to the correct size.

Poser also provides advanced features for lighting and color control. It includes global illumination effects, as well as point lights and spotlights that attenuate naturally over distances. It includes tools for gamma control, tone mapping, and exposure to preserve colors and brightness for post-process rendering. The program's preview capability allows users to view light and color adjustments before fully rendering them for final production. Poser renders characters using a 64-bit engine and provides facilities to render large scenes over a network for increased speed.

Bondware Poser is a solid program for digital art, architecture, film, and video game production. Although it is mainly used by hobbyists, professionals may use Poser for pre-visualization mock-ups and storyboarding, as well as producing high-quality 3D art and animations. Overall, the program lacks an extensive set of advanced design tools for professionals but is a good choice for amateur users.
