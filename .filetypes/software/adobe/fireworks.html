Software Overview

Adobe Fireworks is a bitmap and vector graphics editor. The program was originally developed by Macromedia before being acquired by Adobe in 2005. Adobe then ceased development of the program in May 2013, but it is still available in Creative Cloud as the Creative Suite 6 (CS6) version.

Fireworks is designed to be a lightweight tool for creating graphics for webpages. The application features layer-based editing, smart guides, integration with Photoshop, Illustrator, and Dreamweaver, and image optimization for formats such as JPEG, PNG, and GIF. When you are finished creating your graphics, you can export them to the aforementioned formats, along with TIFF, SWF, BMP, and PDF.

The application allows you to create JavaScript commands that perform certain actions, such as converting a shape color to gray, in order to help you save time on repetitive tasks. You can also add format support, like the ability to import and export the .SVG format, with JavaScript extensions.

Fireworks is not as popular as Photoshop or Illustrator, but the web designers who use it, swear by it. Users find it helpful for UI design, working with vector shapes, and creating live prototypes. However, with its future in doubt, the fuse has been lit on Adobe Fireworks, making Photoshop, Illustrator, or Sketch, better routes for you to take.
