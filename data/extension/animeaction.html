What is an ANIMEACTION file?

An ANIMEACTION file describes a custom action created by Anime Studio, a program used to create animated scenes and movies. It contains JSON text that controls how a character moves in a frame. Saving custom actions as ANIMEACTION files allows animators to re-use their favorite actions across multiple scenes and movies.

More Information

Lost Marble Moho, which was previously known as Anime Studio, is a 2D animation program that allows animators to create animated scenes and movies. Animators can create custom actions that control how a character moves in a frame and then apply them to characters.

In Moho, custom actions are saved and exported as .MOHOACTION files. In Anime Studio, custom actions were saved as ANIMEACTION files. However, the MOHOACTION and ANIMEACTION file formats are similar. With some adjustments, you can continue to use ANIMEACTION files in Moho.

How to open an ANIMEACTION file

You can import an ANIMEACTION file into Lost Marble Moho for use as a custom action. To do so:

  1. Change your ANIMEACTION file's extension to .mohoaction.
  2. Open Moho and then open the Actions window (Window → Actions).
  3. Click the Import Action icon.
  4. Navigate to and select your MOHOACTION file.

The action your file contains will appear in Moho's Actions window.

If necessary, you can also open an ANIMEACTION file in any text editor to view the JSON text it contains. However, you likely should not edit the text your ANIMEACTION contains, as it could cause your action to function incorrectly in Moho.
